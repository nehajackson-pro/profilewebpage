# Personal Showcase Website

A responsive personal portfolio website built with HTML and CSS, featuring Flexbox and Grid layouts.

## Features

- Responsive design
- Flexbox layout for profile section
- CSS Grid for showcase section
- Modern gradient theme
- Hover animations and effects

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts
- Unsplash images for placeholders
